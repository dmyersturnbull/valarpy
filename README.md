# Valarpy

Python code to talk to [Valar](https://github.com/kokellab/valar).

There is more documentation available in the Valar readme, including an [Entityâ€“Relationship Diagram](https://github.com/kokellab/valar/blob/alttables/docs/erd/valar\_schema.png), where the lines are [foreign keys](https://en.wikipedia.org/wiki/Foreign_key) whose targets are denoted as forks.

Core C++ and Python API for running SauronX:  Run a camera, schedule Arduino writes, and record sensor data.
